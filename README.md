# 🧠 Resume Analyzer + AI Tailor

A Flask-based AI resume analysis app powered by OpenAI (via OpenRouter). Upload your resume or paste content, choose a language, and get:

- ✅ Tailored bullet points for specific job descriptions
- ✅ AI-written cover letters
- ✅ Job title + industry matches
- ✅ Resume bullet rewrites
- 🌍 Multi-language support
- 📄 PDF/Text upload support

Built with: Flask, HTML/CSS, JavaScript, OpenAI API, PyMuPDF, jsPDF
